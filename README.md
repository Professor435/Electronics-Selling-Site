# Electronics-Selling-Site
Design Features:
Dark futuristic aesthetic with neon cyan/purple accents
Glass morphism effects and gradient backgrounds
Smooth animations and micro-interactions
Fully responsive design (mobile, tablet, desktop)

Functionality:

Product Catalog with filtering by category (Audio, Charging, Gaming, Smart)
Live Search functionality in the navigation bar
Shopping Cart with add/remove/quantity controls and persistent state
Quick View modal for product details
Flash Deals section with countdown timer
Newsletter subscription form
Smooth scrolling navigation
Interactive Elements:
Hover effects on all product cards with 3D transforms
Animated floating elements in the hero section
Toast notifications for user actions
Loading states and transitions
Mobile hamburger menu
Cart sidebar with overlay
Products Included:
Premium headphones with ANC
High-capacity power banks
Mechanical gaming keyboards
Smart home controllers
Wireless earbuds
Gaming mice
Bundle deals with discount badges

The site is completely self-contained in one HTML file with no external dependencies beyond Tailwind CSS and Font Awesome (loaded via CDN). All JavaScript functionality works immediately including the cart system, filtering, and responsive interactions
